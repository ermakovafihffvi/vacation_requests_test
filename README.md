# Requirements
- Node.js ^20.19.0 or >=22.12.0
- npm ^10.0.0

# How to run
```sh
npm install
```

```sh
npm run dev
```

Project will run on `localhost:5173`, and send `api` requests on `localhost:3000`

# Description
This is a standart `Vue3` project built with `Vite`. It uses `pinia` storage, `PrimeVue`/`Tailwindcss` frontend, and `axios` for sending requests. 
- `api.js` - object with used axios requests
- `stores/` - pinia storages (`users` and `vacations`)
